# Rezervační systém pro Barber Shop

Tento projekt slouží k digitální správě rezervací v kadeřnickém salónu. Aplikace umožňuje zákazníkům vytvořit rezervaci a správci spravovat objednávky přes zabezpečenou administraci.

## Funkce

- Online rezervační formulář s výběrem služeb a barberů.
- Automatická kontrola obsazenosti (systém nepovolí duplicitní termíny).
- Zabezpečená administrace (přihlášení pomocí sessions).
- Přehled rezervací s možností mazání vyřízených záznamů.

## Instalace

1. Naklonujte nebo vložte složku do 'xampp/htdocs/'.
2. Importujte soubor 'barber_shop.sql' do svého phpMyAdminu.
3. Spusťte Apache a MySQL v XAMPP.
4. Otevřete aplikaci v prohlížeči na adrese: 'http://localhost/barbershop/'
